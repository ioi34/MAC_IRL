## 가중치 해석 기준

$w_t = \beta + B C_t$

- `beta`: 모든 컨텍스트가 표준화 평균 0일 때의 기본 보상 가중치
- 컨텍스트 계수 `B`: 해당 컨텍스트가 1표준편차 높아질 때 가중치 변화
- 환율수익률 양수: USD/KRW 상승, 즉 원화 약세
- 양의 가중치: 해당 신호가 양수일 때 매수 방향
- 음의 가중치: 해당 신호가 양수일 때 매도 방향

통합 모델은 기준선보다 NLL이 나쁘므로 아래 결과는 주 결론이 아닌 탐색적 해석으로 사용한다.
