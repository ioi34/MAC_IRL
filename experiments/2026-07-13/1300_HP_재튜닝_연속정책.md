# 하이퍼파라미터 재튜닝 (연속 정책용) — 최종 HP 확정

- 날짜/시간: 2026-07-13 13:00
- 목적: 정책함수가 이진 softmax → 연속(가우시안/MSE)으로 바뀐 뒤에도 투자자별 HP를 이진용 값 그대로 재사용해 왔음. 최종 5특징이 확정된 지금, 연속 목적함수에 맞게 HP(epochs·lr·λ·batch) 재튜닝.
- 방법: 최종 5특징(underwater·momentum·relative·herd·shortmom_orth) 위에서 투자자별 그리드(lr×epochs×λ) 서치를 **15 split**(과적합·시간 절감)로 → 상위 구성을 **전체 45 split**로 확인. batch_size는 256으로 통일.
- 고정 조건: κ=1, CPCV, split별 train-only 스케일링, seed 42. numpy 재현.
- 결과 폴더: `runs/continuous_final_tuned/beta_weights.csv`, 스크립트 `/tmp/hptune.py`, `/tmp/final_tuned.py`.
- 주요 결과: **외국인만 의미 있게 개선(l1 0→0.005), 개인·기관은 기존 HP가 이미 적정. 구조 해석 불변.**

- 해석:
  - 이진용 HP가 대체로 적정했음 — 개인·기관은 재튜닝해도 사실상 불변.
  - 유일한 실질 개선은 **외국인 L1 정규화 추가(λ 0→0.005)**: 상관 0.259→0.272, 약했던 underwater 가중치가 더 눌려 외국인 보상함수가 깨끗해짐.
  - **부호·순위·100% 일관성이 모두 유지** → 복원된 보상구조(외국인 추세추종 / 개인 역추세·처분효과 / 기관 무선호)가 HP에 강건함(강건성 증거).
- 주의: 그리드를 15 split로 튜닝 후 45 split로 확인 → 같은 CV 계열이라 외국인 +0.013 상관 개선은 다소 낙관적일 수 있음. 다만 변화가 작고 "정규화 추가"라는 방향이 이론적으로 타당.
