# Parkinson volatility(magnitude) 재추가 — 기각

- 날짜/시간: 2026-07-13 12:00
- 목적: "연속 모델에선 레벨 특징도 방향적 순flow 효과가 있으면 살 수 있다"는 여지에 따라, 현재 5특징 최종본에 Parkinson을 **magnitude(크기, 항상 양수)** 형태로 재추가해 실증 확인. (vkospi·illiquidity는 논외.)
- 변경한 것: baseline(underwater·momentum·relative·herd·shortmom_orth)에 pk_mag = sqrt( mean_{20d}( (ln(H/L))² / (4 ln2) ) ) 를 feature로 추가.
- 고정 조건: κ=1, CPCV 45 split, split별 train-only 스케일링, 투자자별 HP, seed 42. numpy 재현.
- 결과 폴더: 스크립트 `/tmp/pk_feature.py` 재현 가능.
- 주요 결과: **기각.** 외국인 −0.021/−0.021, 개인 방향 −0.009. 기관만 +0.014/+0.030(허위 신호 부풀림).

- 해석:
  - magnitude 형태도 **외국인(최선 설명 주체)을 −0.021 깎음**. 외국인 β −0.025(91%)는 "고변동 시 순매도(디리스킹)"라는 부호를 갖지만 OOS로 일반화되지 않음.
  - 유일한 "개선"인 기관(+0.014/+0.030)은 원래 6특징 ablation에서 본 **volatility→기관 허위 신호 부풀림**의 재현(기관 상관 0.071은 여전히 무의미). 진짜 선호 신호 아님.
  - 선별에서 방향 내용 ≈0(외국인 −0.064 등) + VKOSPI와 0.606 중복 → 결과가 정확히 예고대로.
- 결론: volatility는 **방향형(ablation 제거)·magnitude(본 실험 기각)** 양쪽 모두 검증 완료 → 완전 종결. 최종 모델 5특징 유지.
