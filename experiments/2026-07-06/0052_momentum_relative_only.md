# momentum + relative 2피처 축소 모델

- 날짜/시간: 2026-07-06 00:52 KST
- 목적: `merged_e2`의 6개 보상특징 중 `momentum`, `relative`만 남겨도 세 투자자의 예측 성능이 유지되는지 직접 검증
- 변경한 것: 보상특징을 `momentum + relative`로 축소
- 고정 조건: `kospi_return_1d + fx_level_z_252` 컨텍스트, 투자자별 `merged_e2` HP, 973표본, CPCV 10-fold/2-test-fold 45 split, purge 1, embargo 5, seed 42
- 데이터: `samsung_macirl_EXTENDED_V2019_2025.csv`, 2022-01-06~2025-12-29
- 설정 파일: `experiments/2026-07-06/configs/momentum_relative_only/`
- 결과 폴더: `runs/momentum_relative_only/`
- 주요 결과: 2피처 모델은 전체 평균에서 6피처 기준선보다 Accuracy와 Macro F1이 높고 NLL이 낮았다. 투자자별로는 외국인이 개선됐고, 기관·개인은 지표별로 비슷하거나 소폭 악화됐다. 모든 OOF 차이의 95% bootstrap CI가 0을 포함해 두 모델의 성능 차이는 통계적으로 구분되지 않았다.

- 해석:
  - 외국인은 4개 특징을 제거했는데도 세 지표가 모두 개선됐다. 현재 데이터에서는 momentum과 relative가 외국인 예측의 핵심 축이고 나머지 특징은 성능상 노이즈일 가능성이 있다.
  - 기관은 Accuracy·NLL이 미세하게 악화됐지만 Macro F1은 개선됐다. 차이가 매우 작고 bootstrap CI도 0을 포함한다.
  - 개인은 세 지표가 소폭 악화됐지만 통계적으로 구분되지 않는다. 예측 성능만 보면 2피처 축소를 배제할 근거는 없다.
  - 따라서 “2피처 모델이 6피처 모델과 비슷한 성능을 낸다”는 해석은 실제 재학습 결과로 지지된다. 다만 보상 해석 범위를 momentum·relative로 축소하는 대가가 있다.
