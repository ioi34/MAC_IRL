# 연속 정책 베이스라인 (merged_e2 6특징 미러링)

- 날짜/시간: 2026-07-12 17:00
- 목적: 이진 최종(merged_e2)을 그대로 옮긴 6특징 구성으로 **연속(강도) 정책의 기준점**을 확보. 이후 제거/추가 판단의 근거 베이스라인.
- 변경한 것:
  - 연속 특징 구성을 임의 4개(pooled herd, relative 없음)에서 **merged_e2 6개**로 미러링: underwater·momentum(20d)·relative(20d)·volatility(20d Parkinson 방향형)·herd_a·herd_b(pairwise). config: `configs/features_continuous_mirror.yaml`.
  - 컨텍스트 2개(kospi_return_1d·fx_level_z_252, 완전 상호작용 → B 12개).
  - 데이터 헤더 공백 오염(`'date      '`) 정리 후 재생성.
- 고정 조건: κ=1, 가우시안 정책 → 예측 = clip(β·φ + Σ_c C_c(φ·W[:,c]), −1,1). 손실 = MSE(clip, 행동)+λ·L1. CPCV 10/2/purge1/embargo5(45 split), split별 train-only StandardScaler(특징·컨텍스트), 투자자별 HP(이진용 재사용), seed 42.
- 데이터: `data/raw/samsung_macirl_EXTENDED_2019_2025.csv`(1,719일, vkospi·semi 포함), 표본 2022-01~2025-12 973일. processed: `data/processed/dataset_continuous_mirror.npz`.
- 설정 파일: `configs/{data_continuous, features_continuous_mirror, model, train, experiment_continuous}.yaml`.
- 결과 폴더: `runs/continuous_mirror_repro/` (cv_metrics.csv, beta_weights.csv, B_weights.csv).
- 주요 결과:
  - **외국인·개인은 설명되고 기관은 무신호** — 이진과 동일 패턴이 연속에서도 재현.
  - 방향정확도 외국인 0.587 / 개인 0.594 / 기관 0.510. 강도상관(실제 vs 예측) 외국인 0.208 / 개인 0.232 / 기관 0.055.
  - **포화율 0** → clip이 한 번도 안 걸림. 행동(u) 크기가 작아 강도 모델이 사실상 선형. (강도 자체가 크지 않음)
  - 기관 β 전부 ≈0(momentum·relative·herd_a 부호일관성 50~78%) → **기관은 연속 강도에서도 선호-설명 불가**, 앞선 발견 확증.

- 해석: 이진→연속 정책 하나만 바꾼 결과, 방향정확도·부호 패턴이 merged_e2와 거의 동일(외국인 momentum+/relative+/herd−, 개인 relative−/momentum−/herd−/underwater+). 연속은 여기에 **강도상관**을 추가로 제공(외국인·개인 ~0.2, 기관 ~0.05). 포화율 0은 강도 크기가 작아 clip이 무의미함을 뜻함.
- **재현 주의**: torch가 샌드박스 프록시 차단으로 미설치 → 저장소 CPCV·스케일러는 그대로 쓰고 모델+Adam만 numpy로 재현. 수식 등가이나 최적화 RNG 차이로 torch 실행과 소수 셋째 자리 수준 차이 가능. 정확한 저장소 숫자는 사용자 Mac(.venv torch)에서 `scripts/train_continuous.py --features-config configs/features_continuous_mirror.yaml`로 재확인 권장.
