# 연속 베이스라인 Ablation — 특징 제거 판정

- 날짜/시간: 2026-07-12 17:30
- 목적: 연속 6특징 베이스라인에서 각 특징의 예측 기여를 정량화해 **제거할 특징을 증거로 결정**(교수님 검증축 2). 제거는 전 주체 공통(global).
- 변경한 것: leave-one-out 6개 + 그룹 2개(행동재무군 underwater+herd_a+herd_b / 전통군 momentum+relative). 각 구성 재학습 후 주체별 방향정확도·강도상관 하락폭.
- 고정 조건: 베이스라인과 동일(κ=1, CPCV 45 split, split별 train-only 스케일링, 투자자별 HP, seed 42). numpy 재현.
- 데이터: `data/processed/dataset_continuous_mirror.npz` (973일, 6특징).
- 결과 폴더: `runs/continuous_mirror_repro/ablation.csv`.
- 주요 결과:
  - **volatility = 제거 대상(확실).** 외국인 방향 −0.029·강도상관 −0.031 → 빼면 외국인이 0.587→0.616, corr 0.208→0.240로 **오히려 개선**. 개인엔 0, 기관의 허위 신호(0.05→0.02)만 부풀림. 전형적 "장식+방해" 특징.
  - **underwater = 경계선.** 외국인엔 방해(−0.013/−0.026), 개인엔 약기여(+0.006/+0.004). 순효과 약함 → 제거 후보.
  - **momentum·relative = 핵심 유지.** 그룹 제거 시 외국인 방향 +0.042·강도상관 +0.075, 개인 +0.022/+0.039 붕괴. 전통(추세)군이 설명력의 엔진.
  - **herd_b(외국인)·herd_a(개인) 약기여**하나 −0.82 공선성 → pairwise vs pooled 비교 필요.
  - 행동재무군 통째 제거 시 외국인 오히려 개선(방향 −0.008·강도상관 −0.019) → 외국인에겐 행동군이 거의 장식.
