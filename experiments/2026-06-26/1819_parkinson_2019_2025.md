# Parkinson 변동성 + 2019-2025 데이터 실험

- 날짜/시간: 2026-06-26 18:19
- 목적: close-to-close 변동성을 Parkinson 방식으로 교체하고, 2019-2025 확장 데이터(고/저가 포함)로 성능 변화 확인
- 변경한 것:

- 데이터: `data/raw/samsung_macirl_EXTENDED_2019_2025.csv` (2019-01-02 ~ 2025-12-29)

## 해석

- Parkinson 적용으로 외국인 volatility 일관성이 0.600 → 0.889로 크게 향상
- 기관 volatility 일관성 1.000 달성: 변동성 클 때 매수를 매우 일관되게 선호 (저가매수)
- 개인 volatility는 여전히 일관성이 낮음 (0.756)
- persist 피처가 개인(0.109)과 기관(0.058) 모두에서 유효한 신호 확인
- 기관 accuracy가 2021 데이터 대비 0.527 → 0.541로 향상 (더 많은 학습 데이터 효과)
