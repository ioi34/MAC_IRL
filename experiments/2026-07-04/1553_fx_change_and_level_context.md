# 환율 변화 보상특징과 환율 수준 컨텍스트 비교

> 최종 채택 결과는 `experiments/2026-07-04/1649_final_reward_context_selection.md`를 따른다.

- 날짜/시간: 2026-07-04 15:53 KST
- 목적: 환율 변화의 직접 보상효과와 환율 수준의 컨텍스트 조절효과를 분리하고, 각각 및 동시 사용의 해석 안정성과 CPCV 성능 비교
- 변경한 것:
  - 보상특징: `fx_sensitivity_1d = action × (-log(USDKRW_t / USDKRW_{t-1}))`
  - 컨텍스트: `fx_level_z_252 = (log(USDKRW_t) - 과거 252일 log 환율 평균) / 과거 252일 표준편차`
  - 수준 통계는 `t-1`까지로 계산해 현재 수준과 과거 기준선을 비교
- 고정 조건: underwater, herd(5), momentum(20), Parkinson volatility(20), persist 제외, KOSPI200 1일 컨텍스트 유지, binary 행동, 973표본, CPCV 10-fold/2-test-fold 45 split, purge 1, embargo 5, seed 42, 200 epochs, batch 256, learning rate 0.01, L1 0.001
- 데이터: `samsung_macirl_EXTENDED_2019_2025.csv`, 2022-01-06~2025-12-29
- 설정 파일: `experiments/2026-07-04/configs/features_fx_semantics_*.yaml`, `experiments/2026-07-04/configs/fx_semantics_e*.yaml`
- 결과 폴더: `runs/fx_semantics_e0_no_fx/`, `runs/fx_semantics_e1_level_context/`, `runs/fx_semantics_e2_change_reward/`, `runs/fx_semantics_e3_both/`
- 주요 결과: 기관은 환율 변화 보상특징, 개인은 환율 수준 컨텍스트에서 가장 분명한 개선이 나타났다. 외국인은 환율 없음이 가장 안정적이었다. 동시 모형의 전체 Macro F1은 기준선과 거의 같고 NLL은 0.004336 악화했지만, 서로 다른 환율 경로의 계수 방향은 대체로 유지됐다.

- 해석:
  - 기관의 환율 변화 beta는 단독과 동시 모형에서 각각 +0.066466, +0.071425이며 부호 일관성이 모두 100%다. 원화 약세 시 기관의 매도 방향 보상이 커지는 패턴이다.
  - 개인은 환율 수준 컨텍스트에서 NLL이 0.010004 낮아지고, 45개 split 중 68.9%에서 NLL이 개선됐다. 고환율 국면에서 개인의 underwater·herd·momentum 반응이 달라지는 효과가 안정적이다.
  - 외국인은 환율 수준 컨텍스트를 추가하면 세 성능지표가 모두 악화했다. 안정적인 B 부호가 곧 일반화되는 유용한 메커니즘을 뜻하지는 않는다.
  - 동시 모형에서도 기관의 직접 변화 beta와 주요 수준 B의 방향은 유지되어 두 환율 변수는 통계적으로 완전히 같은 신호가 아니다. 다만 모든 투자자에게 동시 적용하면 복잡도 증가 대비 전체 성능이 나빠진다.
- 결론:
  - 경제적 변수 체계에는 `환율 변화 보상특징`과 `환율 수준 컨텍스트`를 서로 다른 개념으로 유지할 수 있다.
  - 공통 주모형에 둘을 무조건 동시에 넣는 것은 채택하지 않는다.
  - 해석 우선 최종 보고에서는 공통 ablation을 유지하고, 투자자별 근거는 `기관=변화 보상특징`, `개인=수준 컨텍스트`, `외국인=직접적인 환율 효과 근거 없음`으로 제시한다.
  - 모든 투자자가 같은 입력 구조를 가져야 한다면 변화 보상특징을 주모형에 두고, 수준 컨텍스트는 확장 모형으로 보고하는 구성이 가장 단순하다.
- 다음 액션: 신경망에서도 환율 변화는 reward input, 환율 수준은 context input으로 분리하되, 공통 선형 주모형과 위 네 ablation을 해석 기준선으로 유지
