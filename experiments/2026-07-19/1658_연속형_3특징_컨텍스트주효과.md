# 연속형 3특징 + 컨텍스트 주효과(α) 모델 실험

- 날짜/시간: 2026-07-19 16:58 (Asia/Seoul)
- 목적: 1720 진단(컨텍스트 주효과 부재가 1순위 문제)에 따라, 정책을 `a_t = clip(αᵀC_t + (β + B·C_t)ᵀ x_t)`로 확장하고 3특징(momentum, herd, underwater) 모델에서 컨텍스트 주효과의 기여를 검증한다.
- 변경한 것 (코드):
  - `src/models/continuous.py`: `context_main_effect` 옵션 추가. 활성화 시 컨텍스트 수 크기의 파라미터 `context_main`(α)이 state score에 `αᵀC_t`로 더해지고 L1 페널티에 포함된다. 기본값 false로 기존 동작 불변.
  - `scripts/train_continuous.py`: `model.context_main_effect` 플래그 연결, split별 α를 `context_main_weights.csv`/`context_main_weights_summary.csv`로 저장, `cv_summary.json`에 포함.
  - `src/evaluation/interpret.py`: `summarize_context_main_weights` 추가.
  - `src/training/continuous_trainer.py`: weight history에 `alpha` 행 기록.
  - `tests/test_continuous_behavior.py`: 주효과 state score·L1 검증 테스트 추가. 전체 56개 테스트 통과.
- 변경한 것 (실험): 두 변형 실행 — ① 기본 컨텍스트(kospi_return_1d, fx_level_z_252) + 주효과, ② vkospi_1d 단독 + 주효과.
- 고정 조건: 같은 날 1625/1720 실험과 동일 — 3특징, `κ=1`, seed 42, CPCV 10/2/purge1/embargo5 45 split, split별 train-only 표준화(컨텍스트 포함), 투자자별 HP 동일.
- 데이터: `data/processed/dataset_continuous_reward3_vkospi.npz` 973행(2022-01-06~2025-12-29). 동일 표본.
- 설정 파일:
  - `configs/features_continuous_reward3_vkospi.yaml`
  - `configs/experiment_continuous_reward3_ctxmain_default.yaml`
  - `configs/experiment_continuous_reward3_ctxmain_vkospi.yaml`
- 결과 폴더:
  - `runs/continuous_reward3_ctxmain_default/` (기본 컨텍스트 + α)
  - `runs/continuous_reward3_ctxmain_vkospi/` (vkospi + α)
- 그래프 (`experiments/2026-07-19/1658_continuous_reward3_ctxmain_graphs/`):
  - `comparison_ctxmain_oos.png` — 5변형 OOS 비교(컨텍스트 없음 / B만 2종 / α 포함 2종)
  - `context_main_terms.png` — α 계수와 부호 일관성
  - `ctxmain_default/`, `ctxmain_vkospi/` — 실행별 표준 그래프 3종
- 주요 결과: **주효과 α를 넣자 컨텍스트가 처음으로 성능에 양(+)의 기여를 했다.** 상호작용(B)만 있을 때는 컨텍스트가 성능을 깎거나 무효였지만, α 추가 후 기관 방향정확도 0.525→0.542, 개인 방향 0.601→0.608·상관 0.234→0.245(기본+α), 외국인 방향 0.634→0.640(vkospi+α)으로 컨텍스트 없음 기준선을 전 투자자에서 상회하는 변형이 생겼다. α 계수는 전 항목 93~100% 부호 일관성으로 B 상호작용보다 훨씬 안정적이다. 다만 개선폭은 여전히 split 표준편차 이내다.

- 해석:
  - α의 부호는 1720 진단에서 본 직접 상관과 정확히 일치한다: 외국인은 KOSPI 상승일 순매수(+0.036)·원화 약세 국면 순매도(−0.026)·고변동성 국면 순매도(−0.030), 개인은 정확히 그 반대(KOSPI 하락일 매수, 원화 약세 국면 매수, 고변동성 매수). 개인이 외국인 매도 물량을 받아내는 구조와 정합적이다.
  - 상호작용 B만으로는 담지 못하던 "수준 정보"를 α가 흡수하면서, B만 있을 때 성능을 깎던 기본 컨텍스트(개인 방향 0.590)가 α 포함 시 최고 성능(0.608)으로 뒤집혔다. 진단 1(주효과 부재)이 실제 병목이었음을 지지한다.
  - 외국인 `underwater` β가 주효과 추가 후 +0.0094(95.6%)로 살아났다. B만 있을 때 96~100% 일관성이던 컨텍스트 노이즈 흡수가 줄어든 효과로 보인다.
  - 다만 최고 개선폭(기관 +0.017)도 split 표준편차(0.038)의 절반 이하라 단일 seed 기준으로는 잠정적이다. 투자자별 최적 컨텍스트가 다른 점(외국인 vkospi, 기관·개인 기본 2종)도 추가 확인이 필요하다.
