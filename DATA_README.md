# 단일종목 MAC-IRL 데이터셋 (삼성전자 005930)

삼성전자 일별 수급·시세·거시 컨텍스트 데이터셋. 외국인·기관·개인의 매매 행동으로부터 잠재 보상 함수를 추정하는 MAC-IRL 연구의 입력 데이터입니다.

- **종목**: 삼성전자 (005930, KOSPI)
- **빈도**: 일별 (거래일)
- **기간**: 2021-01-04 ~ 2025-12-30
- **행 수**: 1,225 거래일 (결측치 없음)
- **파일**: `samsung_macirl_2021_2025_FINAL.csv` (UTF-8-SIG), `.xlsx`

## 컬럼 정의

| 컬럼명 (영문) | 한국어 | 단위 | 설명 |
|---|---|---|---|
| `date` | 날짜 | YYYY-MM-DD | 거래일 |
| `close` | 종가 | 원 | 수정주가 미적용(실제 종가) |
| `return` | 종목 수익률 | 비율(소수) | `close / 전일 close − 1` (예: 0.0247 = +2.47%) |
| `trading_value` | 거래대금 | 원 | KRX 실제 거래대금 |
| `foreign_net_buy` | 외국인 순매수 | 원 | 거래대금 기준, 외국인 합계 |
| `institution_net_buy` | 기관 순매수 | 원 | 거래대금 기준, 기관 합계 |
| `retail_net_buy` | 개인 순매수 | 원 | 거래대금 기준 |
| `usdkrw` | 원/달러 환율 | 원 | 종가 |
| `kospi200_return` | KOSPI200 수익률 | 비율(소수) | `KOSPI200 종가 / 전일 종가 − 1` |

순매수 부호: 양수 = 순매수(매수 우위), 음수 = 순매도(매도 우위). 네 주체(외국인·기관·개인·기타법인)의 순매수 합은 0이 되도록 정의됨(기타법인은 본 데이터셋에서 제외).

## 데이터 출처

| 변수 | 출처 |
|---|---|
| `close`, `return`, `trading_value` | KRX 데이터 마켓플레이스 — 개별종목 시세 추이 [12003] |
| `foreign/institution/retail_net_buy` | KRX 데이터 마켓플레이스 — 투자자별 거래실적(개별종목) [12008], 일별추이·거래대금·순매수 |
| `usdkrw`, `kospi200_return` | FinanceDataReader (USD/KRW, KS200) |

> KRX는 2025-12-27 회원제(KRX 데이터 마켓플레이스)로 전환되어 로그인이 필요합니다. 시세·투자자 데이터는 로그인 후 개별종목 단위로 연도별 xlsx 다운로드하여 병합했습니다. (CSV 다운로드는 한글 인코딩이 깨져 xlsx 사용.)

## 수집·병합 방법

1. KRX에서 005930의 일별 시세(종가·거래대금)와 투자자별 순매수(거래대금)를 연도별(2021~2025) xlsx로 다운로드.
2. FinanceDataReader로 USD/KRW와 KOSPI200 지수를 받아 수익률 계산.
3. 모든 소스를 `date` 기준으로 병합. 환율은 거래일 기준 결측 시 직전값으로 보정.
4. `return`은 2020-12월 버퍼를 포함해 계산하여 2021년 첫 거래일도 결측이 없도록 처리.

## 사용 참고 (연구 설계)

- 순매수 정규화: `u_i,t = net_buy_i,t / trading_value_t`.
- 행동 라벨(초기 3분류): `u`의 분위수로 하위 30% = sell(-1), 중간 40% = neutral(0), 상위 30% = buy(+1).
- 시점 누수 방지: 예측 구성 시 입력 `s_t, C_t` / 라벨 `a_{t+1}`로 한 시점 밀어 사용.
