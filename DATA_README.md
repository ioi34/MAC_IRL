## 컬럼 정의

### 기본 변수 (연구 설계 필수)

| 컬럼명 | 한국어 | 단위 | 설명 |
|---|---|---|---|
| `date` | 날짜 | YYYY-MM-DD | 거래일 |
| `close` | 종가 | 원 | 수정주가 미적용(실제 종가) |
| `return` | 종목 수익률 | 비율 | `close/전일close − 1` |
| `trading_value` | 거래대금 | 원 | KRX 실제 거래대금 |

### 확장 변수 (투자자별 매수·매도 분리)

`{i}` = `foreign`(외국인) · `institution`(기관) · `retail`(개인) 각각에 대해:

| 컬럼명 | 한국어 | 단위 | 출처 |
|---|---|---|---|
| `{i}_buy_value` | 매수금액 | 원 | KRX 직접 다운로드 |
| `{i}_sell_value` | 매도금액 | 원 | 파생: `buy_value − net_buy` |
| `{i}_buy_volume` | 매수수량 | 주 | KRX 직접 다운로드 |
| `{i}_sell_volume` | 매도수량 | 주 | KRX 직접 다운로드 |

관계식: `net_buy = buy_value − sell_value`, `평균매수가 ≈ buy_value / buy_volume`, `평균매도가 ≈ sell_value / sell_volume`. 이 변수들로 투자자별 평단(평균단가) 갱신, 보유수량 proxy, 손실실현/물타기 feature를 정밀하게 구성할 수 있습니다.

- 1,225행, 결측치 0, 날짜 중복 0, 오름차순 정렬, 주말 없음.
- `return`을 `close`로 재계산 → 컬럼값과 오차 0.
- KRX 종가 vs FinanceDataReader 종가 100% 일치.
- 매도금액·매도수량 음수 0건. `buy_value − sell_value`가 기존 순매수와 오차 0으로 일치.
- 매수/매도 방향 판별: 연도별 부호 일치율이 모두 ~100% 또는 ~0%로 명확히 분리(애매한 해 없음).
- 교차검증: 평균매수가(`buy_value/buy_volume`)가 외국인·기관·개인 모두 종가 대비 평균 0.4% 이내.

## 사용 참고 (연구 설계)

- 순매수 정규화: `u_i,t = net_buy_i,t / trading_value_t`.
- 행동 라벨(3분류): `u`의 분위수로 하위30% sell(−1) / 중간40% neutral(0) / 상위30% buy(+1).
- 시점 누수 방지: 입력 `s_t, C_t` / 라벨 `a_{t+1}`로 한 시점 밀어 사용.
- 평단·보유 proxy: 매수수량·매도수량 누적으로 보유수량 근사, 매수금액/매수수량 누적가중으로 평균단가 근사.
